# Aplicación Node.js en Heroku

Esta es una aplicación web simple creada con Node.js y Express y desplegada en Heroku.

## Tecnologías utilizadas

- Node.js
- Express
- Heroku
- GitHub

## Instalación

Clonar el repositorio:

git clone https://github.com/USUARIO/mi-app-heroku.git

Entrar al proyecto:

cd mi-app-heroku

Instalar dependencias:

npm install

Ejecutar el servidor:

npm start

## Rutas disponibles

/ → Página principal  
/about → Información de la aplicación

## Deployment

La aplicación fue desplegada usando Heroku conectado a GitHub.
